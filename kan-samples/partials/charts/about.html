Kaltura Analytics 2.0 Charts

About Kaltura Analytics 2.0 charts implementation

Kaltura Analytics 2.0 uses D3 library to create and manipulate charts.

D3 library allows you to bind arbitrary data to a Document Object Model (DOM). You can use D3 to create an interactive SVG bar chart with smooth transitions and interaction.

D3 is huge library not aiming to charts only. We therefore use NVD3 library. NVD3 is implementation on-top of D# that create re-usable charts and chart components for d3.js without taking away the power of d3.js.

NVD3 is written in pure javascript while our application uses Angular.JS. To have a natural experience we use Angular-NVD3 library.

Where to start?
  • Explore Kaltura Analytics 2.0 charts using the side navigation bar.
  • Play with angular-nv3 superb live edit section to discover more features that can be added.
  • Since we are using angular-nvd3 and this library uses json based configuration, you can modify our charts configuration with anything you see in the live edit section and it will affect our application charting.
