Kaltura Analytics 2.0 Maps

About Kaltura Analytics 2.0 maps implementation

Kaltura Analytics 2.0 uses LeafLet library together with Open street map to show map with annotations to the user.

Leaflet is the leading open-source JavaScript library for friendly interactive maps. Leaflet is designed with simplicity, performance and usability in mind. It works efficiently across all major desktop and mobile platforms.

Leaflet is written in pure javascript while our application uses Angular.JS. To have a natural experience we use angular-leaflet-directive library.
