Examples for map charts supported in Kaltura Analytics 2.0 as described in issue #PLAT-4901

This sample show audience annotation according to geographical information.

  • This component load the map from Kaltura dedicated server.
  • There are two types of annotations according to the zoom the user is currently using: Cities when zoomed-in, Countries when zoomed-out.
  • Each annotation is filled with color according to the number of audience as shown in the legend.
